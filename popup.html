Faceit Position Caller is part of the Faceit Stats Provider project.

The extension is in beta and will remain so as long as Faceit developers provide the necessary tools for us, as developers, to ensure our applications work seamlessly with their servers. For now, in the beta version, there are many things that might not work as expected. The best course of action if something goes wrong is to log out and log back in. If that doesn’t resolve the issue, feel free to contact me via Steam.

How does it work?

Entering a message in the first two options of the drop-down menu will send a message to both General and Team Faceit chat as soon as the match starts. The remaining options, which include map names, will automatically send a message to Team chat right after the map is picked.
